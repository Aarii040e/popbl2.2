*********************Elkar-Ekin*********************

* Descripción
Elkar-Ekin es una aplicación desarrollada en SpringBoot que proporciona una plataforma para la colaboración y gestión de una red de voluntarios. La aplicación permite a los usuarios crear, asignar y gestionar tareas de manera eficiente.

Nota:
Se recomienda ejecutar el script elkarrekin.sql que se encuentra en la carpeta para tener contenido creado en la aplicación. Entre el contenido que crea se encuentran los usuarios que son los siguientes:

Usuario: maialen	Contraseña:maialen	Perfil: cliente	
Usuario: amaia		Contraseña:amaia	Perfil: cliente	
Usuario: lide		Contraseña:lide		Perfil: voluntario
Usuario: paula		Contraseña:paula	Perfil: cliente
Usuario: aritz		Contraseña:aritz	Perfil: voluntario
Usuario: admin		Contraseña:admin	Perfil: administrador
